# 


## Setup inicial

Antes de realizar qualquer ação, realizar o setup inicial do projeto

```sh
$ composer install
$ composer dump-autoload
```

## Execução local

Para executar local, rodar o seguinte comando abaixo:

```sh
$ docker compose up
```

## Execução de testes

Para executar os testes, rodar o seguinte comando abaixo:

```sh
./vendor/bin/phpunit
```
